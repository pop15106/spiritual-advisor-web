## 🚀 快速開始

### 步驟一：啟動後端服務

1. 開啟終端機（Terminal）
2. 進入後端資料夾：
   ```bash
   cd c:\Users\7010\.gemini\antigravity\scratch\spiritual_ai_advisor
   ```
3. 如果沒有安裝依賴套件，先執行安裝：
   ```bash
   pip install flask flask-cors python-dotenv
   ```
4. 啟動後端 API：
   ```bash
   python api.py
   ```
5. 看到「🔮 Starting Spiritual AI Advisor API...」表示成功啟動
6. 後端運行在 `http://localhost:5000`

### 步驟二：啟動前端網站

1. 開啟另一個終端機
2. 進入前端資料夾：
   ```bash
   cd c:\Users\7010\.gemini\antigravity\scratch\spiritual-advisor-web
   ```
3. 如果沒有安裝依賴套件，先執行：
   ```bash
   npm install
   ```
4. 啟動前端：
   ```bash
   npm run dev
   ```
5. 看到「✓ Ready」表示成功
6. 前端運行在 `http://localhost:3000`

### 步驟三：開始使用

1. 開啟瀏覽器
2. 輸入網址：`http://localhost:3000`
3. 開始探索您的命運藍圖！

## 📖 各服務使用方式

### 🃏 塔羅占卜

1. 點選首頁的「塔羅占卜」卡片
2. 在心中默想您的問題
3. 可以在輸入框輸入問題（選填）
4. 點選「✨ 抽取三張牌」
5. 系統會抽出三張牌，分別代表：
   - **過去**：影響您現在處境的過去能量
   - **現在**：您目前的狀態與挑戰
   - **未來**：可能的發展方向與指引
6. 每張牌會顯示：
   - 牌名與正/逆位
   - 牌義解讀
   - 關鍵詞標籤

### ☯️ 八字命理

1. 點選「八字命理」
2. 輸入您的出生資料：
   - 出生日期（國曆）
   - 出生時辰
   - 性別
3. 點選「🔮 排盤分析」
4. 系統會顯示：
   - 四柱八字（年柱、月柱、日柱、時柱）
   - 流年運勢走勢圖
   - 各年度運勢提示

### 🧬 人類圖

1. 點選「人類圖」
2. 系統自動載入您的人類圖資料
3. 會看到：
   - 人體圖（Bodygraph）- 顯示9個能量中心
   - 您的類型（如：生產者）
   - 人生策略與人生角色
   - 各能量中心的狀態說明

### ⭐ 西洋占星

1. 點選「西洋占星」
2. 系統顯示：
   - 太陽星座
   - 上升星座
   - 七大行星位置
   - 星盤解讀

### 💜 紫微斗數

1. 點選「紫微斗數」
2. 系統顯示：
   - 命宮主星
   - 十二宮位分布
   - 命格解析
   - 財運與感情分析

### 🌐 多系統整合分析

1. 點選「多系統整合」
2. 選擇要整合的系統（至少選2個）
3. 輸入您的問題
4. 點選「🔮 開始多系統整合分析」
5. AI 會結合各系統觀點給出：
   - 各系統個別觀點
   - 綜合分析
   - 具體建議

## ❓ 常見問題

### Q: 出現「無法連接後端 API」怎麼辦？

**A:** 請確認後端服務已啟動：
1. 檢查是否有執行 `python api.py`
2. 確認終端機顯示「API running at http://localhost:5000」
3. 如果仍有問題，嘗試重新啟動後端

### Q: 網頁載入很慢？

**A:** 首次載入可能需要幾秒鐘，請耐心等待。如果持續很慢：
1. 嘗試重新整理頁面（F5）
2. 清除瀏覽器快取

### Q: 可以在手機上使用嗎？

**A:** 可以！本平台支援手機瀏覽。只要確保手機與電腦在同一個網路，使用 `http://[電腦IP]:3000` 即可訪問。

### Q: 分析結果準確嗎？

**A:** 本平台僅供娛樂參考，分析結果基於傳統命理學原理與 AI 解讀。重大決定請諮詢專業人士。
